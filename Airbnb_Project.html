Data analysis project using real Airbnb data.

Tools used: Excel, MySQL, Tableau

The purpose of this project is to help you decide what kind of Airbnb do you want to provide for your customers. Where should your Airbnb be located? How many people will it accomodate? What should be the minimum length of stay? How much should you charge? These are all important questions one needs answered before starting his Airbnb business.

Downloading and Cleaning the data

My source for this project was insideairbnb.com. I used the latest data available, last scraped 16th of March 2025. I Downloaded Detailed Listing Data for my data source, and a GeoJSON file to help later visualize our data.

Most of the data cleaning was done in Excel. I removed unnecessary columns, reformated some of the data to make opening the csv file in MySQL possible, removed empty cells/rows, removed null values, and assigned new IDs to my data, as the original range of ID numbers was way out of range for MySQL to interpret. Click here to see all of the files used for this project.

Finding Trends using SQL

Next I used MySQL to query through the data, to find trends we could later visualize. Thanks to the number_of_reviews column we could analyze, which Airbnb listings are most popular in relationship to their location, size, minimum nights stay and price. I Also used SQL to find innacurate data, which would harm the visualization, and removed it. The SQL file is included in the repository.

Visualizing the data using Tableau

I exported the final data table from MySQL and loaded it into Tableau Public along with the geoJSON file i downloaded earlier. I performed and Inner Join on the two files to create a map in tableau for the visualization, along with other graphs to show as much useful data as possible.

Key takeaways from the visualization are:

• The most popular neighbourhoods are Praha 1 and Praha 2.
• Accomodations for 4-5 people are the priciest, but accomodations for 2-3 people are the most popular.
• The longer is the minimum stay, the cheaper is the price per night.
• The most popular neighbourhoods tend to be the most expensive.
• Accomodation for 1 person yields the lowest income and also is in the lowest demand.
